¿Qué hace un DESARROLLADOR?

El término "Desarrollador" proviene del Francés "Développeur", y se aplica a personas que se encargan de desarrollar aplicaciones informáticas, siendo esta la principal disciplina dentro de la informática.

La programación es la principal disciplina dentro de la informática. La función de un programador, además de ser un especialista en informática, este es capaz de elaborar sistemas informáticos complejos, implementarlos y ponerlo a punto con distintos lenguajes de programación.
Un Desarrollador escribe, depura y mantiene el código fuente de un programa, que es todo ese conjunto de instrucciones que ejecuta el hardware de una computadora. Es decir, el programador le dice a la computadora lo que tiene que hacer para llegar a una solución de algún problema.
Por supuesto, antes de comenzar siquiera a escribir, tiene que haber un previo analisis del problema para después elaborar el algoritmo que va a dar solución a dicho problema.

¿Qué tipo de aplicaciones?

Una aplicación es un programa diseñado como herramienta para que los usuarios de la misma realicen una o varias tareas. Tipos de aplicaciones:

APP NATIVAS:

Estás son las que se encuentran instaladas en el Sistema Operativo de los ordenadores, tablets, notebook, smartphones, etc. Se programan con distintos lenguajes de programación, este va a depender del sistema Operativo que este corriendo el dispositivo. Por ejemplo, para Windows se programa con C# con .Net, para MacOS esta switch, para Android JAVA, etc.

WEB APP

Estás aplicaciones se encuentran alojadas en un servidor web y necesitan fundamentalmente de internet y de un navegador para ser utilizadas. La gran ventaja de estas es que se puede acceder a ellas desde cualquier Sistema Operativo y desde cualquier dispositivo. Se desarrollan con lenguajes como HTML, CSS y JavaScript.

WEB APP NATIVAS

Combinación de las dos anteriores, también conocidas como aplicaciones hibridas. Se desarrollan con los mismos lenguajes que las WEB APP permitiendo su uso en diferentes plataformas, pero su diferencia es que estas pueden tener acceso a gran parte de las caracteristicas del harware del dispositivo.

¿Qué servicios presta?

Un programador puede prestar varios servicios como desarrollo, mantenimiendo, y depuración de sistemas informáticos. Puede desembolverse tanto del lado del cliente (FRONT-END) o del lado del servidor (BACK-END)... o ambos en conjunto, como los denominados Full-stack.

Front-end

Hace referencia a aquellas personas que se encargan de maquetar la estructura de las aplicaciones del lado del cliente, diseñar la IU y la parte estética de la aplicación. Los lenguajes que utilizan son HTML, CSS y JavaScript, entre otros.

Back-end

Estas personas se encargan de programar las aplicaciones para decirles como manipular los datos ingresados por el usuario desde el front-end, como acceder y guardar en la base de datos. Manipular y responder peticiones y acciones que realiza el usuario desde el front-end entre otras funciones.
